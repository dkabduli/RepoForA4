Dice Roling Game
-----------------------

This app is a simple dice rolling game.

How to play:
1. Install Docker on your machine.
2. Pull the Docker image from Docker Hub:
   docker pull ykabduli/main.py
3. Run the application:
   docker run -it ykabduli/main.py
